# PaliKeyboard
HOW TO UNINSTALL

Win10

2 Files to uninstall

1.
  Goto:
    Computer\HKEY_LOCAL_MACHINE\SYSTEM\ControlSet001\Control\Keyboard Layouts
      usually last file under this directory

2.
  Goto:
    %windir%
  then search for Pali, and it will show up

  if no search
    C:\Windows\System32
  or
    C:\Windows\SysWOW64
